# tequisa-marketplace

Claude Code plugin marketplace by Denny Leonardo.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [tequisa-toolkit](https://github.com/deleon626/tequisa-toolkit) | Docs loaders, meta-generators, Jina research, cargo quoting, AI image gen |

## Installation

```bash
# 1. Add this marketplace (one-time)
/plugin marketplace add deleon626/tequisa-marketplace

# 2. Install a plugin
/plugin install tequisa-toolkit@tequisa-marketplace --scope user
```
